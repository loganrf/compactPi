# compactPi

compactPi is a project to develop a compact Raspberry Pi Compute Module carrier suitable for ArduPilot applications. It's primarily targeted at marine applications (ArduSub/ArduRover).

# Tenets
These tenets will guide the decision making process across the design cycle and future mainline iterations. Think of them as decisional tie breakers and expressions of the overall design philosophy.
